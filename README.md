# Sistema de Gestión de Viajes

Aplicación desarrollada en Python conectada a Oracle Database
mediante una arquitectura Modelo Vista Controlador (MVC).

El sistema permite la gestión de usuarios, destinos turísticos,
paquetes de viaje y reservas.

## Tecnologías utilizadas

- Python
- Oracle Database
- SQL
- Arquitectura MVC
- Git
- GitHub

## Funcionalidades

- Registro de usuarios
- Inicio de sesión
- Gestión de perfiles
- Consulta de destinos turísticos
- Creación de paquetes personalizados
- Reservas de viajes
- Cancelación de reservas
- Gestión administrativa
